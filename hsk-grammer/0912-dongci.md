# 동사
## 동사의 종류
| 종류   | 설명                       | 예제                  |
| ---- | ------------------------ | ------------------- |
| 동작동사 | 동작, 행위를 나타냄              |                     |
| 상태동사 | 사람 혹은 동물의 심리 나타냄         | 爱 恨 怕 嫌 怀念 喜欢 感谢 怀疑 |
| 관계동사 | 주어와 빈어간 존재하는 어떠한 관계를 나타냄 | 叫 像 是 有 当 做 成为 等于   |
| 능원동사 | 가능, 희망, 필요를 나타냄          |                     |
| 趋向动词 | 동작의 방향을 나타냄              | 来 去 进 出 起来 上去 过来 出去 |

## 동사의 문법 특징

1. 술어로 사용 - 대게 동사는 빈어를 가질 수 있으며, 빈어는 대부분 명사 또는 대사이다.

   1. 일부 동사는 목적어를 가지지 못한다. 대부분 **이합사**이다.

      ```
      去北京旅行    跟朋友见面    生活变了
      *旅行北京    *见面朋友     *变了生活
      ```

   2. **일부 동사는 술어성 빈어(동사성 어구,형용사성 어구, 주술구가 빈어)만을 가진다.**

      ```
      促进发展    进行研究    打算离开
      感到高兴    觉得不错    善于发现
      ```
      그 외에 술어성 빈어를 가지는 동사들

      ```
      加以    得以    足以    给予    给以    予以    准予    感觉  觉得
      认为   以为    开始    决定    继续    试图   企图    希望  懒得
      忙于   乐意
      ```

   3. 일부 동사는 명사성 빈어를 가질 수 있으며, 또한 술어성 빈어를 가질 수 있다.

      ```
      保证质量    展开地图    受到压力
      保证完成    展开调查    受到刺激
      ```

      ```
      擅长  适合  保持  坚持  注意  准备  从事  适应  批准  破坏  请求 推动
      限制  协助  训练  要求  支配  引起  指导  指点  喜欢  讨厌  追求 爱
      怕    求    嫌
      ```

   4. 감각 또는 심리를 표현할 때,  일반적으로 주술구(절) 목적어를 가질 수 있다.

      ```
      他知道你回来。
      我发现他胖了。
      请别打扰病人休息。
      听说他离开公司了。
      ```

      ```
      觉得  认为  以为  同意  记得  忘记  相信  希望  盼望  期望  声明
      喜欢  赞成  反对  支持  指出  避免  代表  代替  担心  批评  表示
      承认   说明   提倡  生怕  意味着  标志着
      ```

2. 대개 동사는 **동태조사**를 추가할 수 있다. 了 着 过

   ```
   看了  调整了  躺着  管理着  听过  抗议过
   ```

3. **심리동사와 일부 조동사는 정도부사의 수식**을 받을 수 있다.

   ```
   很喜欢  非常信任  特别怀念
   太能说  真会做   很愿意去
   ```

## 이합사

**중간에 기타성분을 추가**할 수 있는 동사

### 이합동사의 특징

1. 이합사는 대개 **빈어와 쓰이지 아니한다**.

   ```
   *见面朋友   *毕业大学   *发愁没钱    *帮忙朋友
   和朋友见面  从大学毕业  为没钱发愁  给朋友帮忙
   ```

2. **의문대사, 동량사, 시량사**는 이합동사 사이에 놓여야 하며, 이합사 뒤에 놓을 수 없다.

   ```
   *吵架什么    *操心什么    *聊天儿几次    *上班一天
   吵什么架     操什么心      聊几次天儿     上一天班
   ```

3. 이합동사의 중첩은 일반적으로 **AAB**형식이다.

   ```
   跑跑步    见见面    聊聊天儿    化化妆
   点点头    招招手    伸伸腿      跳跳舞
   ```

4. **동태조사**는 이합사 사이에 놓아야 하며, 이합사 뒤에 놓일 수 없다. 

   ```
   周末，爬了一次山。  *周末，爬山了一次。
   他朝我招着手。     *他朝我挥手着。
   我们以前见过面。   *我们以前见面过。
   ```

5. 일부 이합사는 이합사의 object를 이합사의 verb 앞으로 위치시켜 사용할 수 있다. 

   ```
   我发也理了，信也寄了。
   澡已经洗了，准备睡觉了。
   我们歌也唱了，舞也跳了。
   ```

## 동사의 중첩과 형식

중국어의 동사는 중첩하여 사용이 가능하다. 중첩 후에 동사는 일반적으로 **짧게 지속되는 동작** 혹은 **진행되는 회수가 적은 동작**을 표현하거나 **시도**(try)를 표현한다.

### 중첩의 형식

지속되는 동작을 표현하여 동작의 지속시간이 짧거나, 진행의 회수가 적음을 표시한다.

- AA식(단음절동사)

  ```
  AA/A一A(未发生) - 听听/听一听 들어보자 想想/想一想 생각해보자
  A了A（已发生) - 听了听 들어보건데 想了想 생각해보건데
  ```

- ABAB식(쌍음절동사)

  ```
  ABAB(未发生)    打扫打扫    调整调整
  AB了AB(已发生)  打扫了打扫   调整了调整
  ```

- AABB식 - 서로 상반되거나 동일한 의미의 단음절동사를 중첩한다. 동작의 크기가 큼을 의미한다.

  ```
  吵吵闹闹    打打闹闹    说说笑笑    拉拉扯扯
  搂搂抱抱    蹦蹦跳跳    吃吃喝喝    哭哭啼啼
  摇摇晃晃    来来回回    上上下下    进进出出
  ```

- AAB식 - 이합사의 중첩형식이며, 때로는 주기적이거나 습관적 혹은 한가한 동작행위를 표현한다. 

  ```
  聊聊天儿    养养鸟    上上网    爬爬山   下下棋
  打打拳      散散步    跳跳舞    踢踢球    看看书
  ```

### 주의점

1. 동시에 진행되는 동작에는 중첩 불가

   ```
   *大家又说说，又笑笑。
   *他边看看电视，边听听歌。
   ```

2. 현재 진행되는 동작에는 중첩 불가

   ```
   *他正等等你呢。
   *我听听音乐呢。
   ```

3. 정어로 쓰이는 동사는 중첩이 불가능

   ```
   *你等等的是谁？
   *妈妈做做的菜很好吃。
   ```

4. 보어를 가진 동사는 중첩 불가능

   ```
   *你休息休息一下。
   *再玩玩儿几天吧。
   ```

5. 능원동사는 중첩 불가능

6. 동태조사가 뒤에 붙은 동사는 중첩 불가능

   ```
   *我想想了这个问题。
   *你看看过这本书吗？
   ```

7. 중첩된 동사 뒤에는 수량정어를 가질 수 없다.

   ```
   *我去见见一位朋友。
   *他试了试一双鞋。
   ```

8. 연사구에서는 구와 절을 겸하는 V1 동사는 중첩이 불가능하다.

   ```
   *我去去商店买东西。
   *我帮帮朋友买东西。
   ```

# 능원동사

## 능원동사란?

능원동사는 조동사로 부르기도 하며 **가능**, **희망**, **필요**의 의미를 가진다.

| 의미   | 능원동사          |
| ---- | ------------- |
| 능력   | 会 能           |
| 희망   | 要 想 肯 敢 愿意    |
| 가능   | 能 会 要 得 可能 能够 |
| 필요   | 应 该 得 应该 当然   |
| 허락   | 能 可 得 准 可以    |
| 평가   | 配 值得          |

## 능원동사의 문법특징

1. 능원동사는 뒤에 **동사**, **형용사** 또는 **단문**만 붙일 수 있다.
  ```
  黄山值得游览。
  他会好起来的。
  你能说汉语吗？
  我愿意跟你聊聊。
  ```

2. 능원동사는 개사구와 동시에 쓰일 때, 일반적으로 개사구는 **능원동사의 뒤에 위치한다**.
  ```
  我想向你请教。
  他会对你负责的。
  你要跟我说什么？
  你得把钱还给我。
  ```


3. 능원동사는 일반적으로 **동태조사를 가질 수 없다**.

   ```
   他会开车了。       *他会了开车。 
   你可以进来了。     *你可以了进来。
   ```

4. 능원동사는 중첩할 수 없다.

   ```
   我们应该帮帮他。     *我们应该应该帮帮他。
   你可以帮我改改吗？   *你可以可以帮我改改吗？
   ```

5. 부정을 병렬하여 **의문**을 표시할 수 있다.

   ```
   你能不能帮他？
   我该不该去呢？
   你愿不愿意帮他？/ 你愿意不愿意帮他？
   ```

6. 동사는 대개 不 부정을 사용한다.

   ```
   他不会做饭。
   他不肯帮我。
   他不愿意帮我。
   你不应该这样做。
   ```

## 能 VS 会

| 能                                        | 会                                  |
| ---------------------------------------- | ---------------------------------- |
| 가능을 표현, 문장 끝에 단정적인 말투 **的**와 사용할 수 없다.   | 가능을 표현, **的**와 사용 가능               |
| **주관**적으로 어떠한 능력의 소유를 표현                 | **객관**적으로 어떠한 능력의 소유를 표현           |
| 비교적 **높은 수준**의 능력의 소유를 표현                | **처음 습득**한 어떠한 능력의 소유를 표현          |
| 어떠한 능력의 **회복**을 표현                       | 사물의 필연적인 발전을 추측할 때                 |
| 어떠한 일의 **능숙**을 표현                        | 어떠한 일의 **능숙**을 표현                  |
| 사물의 어떠한 **기능**이나 **용도**를 가지고 있음을 표현할 수 있다. | 사물의 어떠한 기능이나 용도를 가지고 있음을 표현할 수 없다. |
| **조건이 부합**됨을 나타낼 때 사용 가능                 | 조건이 부함됨을 나타낼 때 사용 불가               |

## 能 VS 可以

둘 다 **기능**을 표현한다.

```
喝茶能减肥。
喝茶可以减肥。
```

| 能                       | 例如                        | 可以                           | 例如                             |
| ----------------------- | ------------------------- | ---------------------------- | ------------------------------ |
| 어떠한 것이 **능숙함(잘하는)**을 표현 | 他可真能喝酒。<br>记者通常都很能写。      | 어떠한 것을 **할 수 있는 능력**이 있음을 표현 | 你可以喝酒吗？<br> 我可以写，但写不好。         |
| 어떠한 **객관적인 가능성**을 표현    | 雨这么大，他能来吗？<br> 你说今天风还能停吗？ | 어떠한 객관적인 가능성을 표현 **불가**      | *雨这么大，他可以来吗？<br> *你说今天的风还可以停吗？ |

## 应该 VS 该

| 应该                       | 例如                                    | 该                    | 例如                               |
| ------------------------ | ------------------------------------- | -------------------- | -------------------------------- |
| **가정문의 뒷 문장**에 사용 불가     | *做不完作业，明天应该挨批评了。<br> *一定得快点儿，否则应该迟到了。 | **가정문의 뒷 문장**에 사용 가능 | 做不完作业，明天该挨批评了。<br>一定得快点儿，否则该迟到了。 |
| 也와 사용 가능<br>**又와 사용 불가** | 你太累了，也应该歇歇了。<br>*天阴了，又应该下雨了。          | 也, 又 사용가능            | 你太累了，也该歇歇了。<br>天阴了，又该下雨了。        |
| **감탄문**에 사용 불가           | *你不回去，父母应该多担心啊！<br> *看到你来，他应该多高兴啊！    | 감탄문에 사용 가능           | 你不回去，父母该多担心啊！<br>看到你来，他该多高兴啊！    |

## 得(dei3) VS 得(de2)

| 得(dei3)            | 例如                              | 得(de2)                    | 例如                       |
| ------------------ | ------------------------------- | ------------------------- | ------------------------ |
| 실질적인 수요를 나타냄(필요하다) | 有病得去医院。<br>这事得问问律师。             | 허락을 나타냄, 부정형식에서만 사용 가능하다. | 外人不得进入手术室。<br>军事重地，不得入内！ |
| 확신을 표현(~임에 틀림없다)   | 你偷看他的日记，他准得生气。<br>你看这天儿，明天准得下雨。 |                           |                          |

## 要

1. 소망(~하고 싶다)을 표현한다. **부정으로 不想, 不愿意**를 사용한다.

   ```
   我要学摄影。       
   你要跟我一起去吗？
   我不想学摄影。
   你不愿意跟我一起去吗？
   ```

2. 필요(~해야 한다)를 표현한다**. 부정으로 不用, 不必**를 사용한다. 不要(하지 마라)는 주로 금지 혹은 제지할 때 사용한다.

   ```
   你要把钱还给我！
   你不用还我钱了。
   你不必还我钱了。
   不要随便扔垃圾！
   ```

3. 예측(~할 것이다)을 표현한다. 말미에 **항상 了**가 붙는다.

   ```
   他要回来了。
   又要下雨了。
   他快要回国了。
   我就要毕业了。
   ```

4. 가능(~할 수 있다)을 표현하며 말미에 **的**가 붙을 수 있다. **부정 형식은 不会**를 사용한다.

   ```
   看样子又要下雨。
   不抓紧要迟到的。
   时间还早，不会迟到。
   ```

5. 추측(~인 것 같다)을 표현하며, **비교문**에 사용된다. 要는 比자 앞이나 뒤로 위치할 수 있으며, 또한 得의 뒤에 위치할 수 있다.

   ```
   他要比我说得好些。
   他比我要说得好些。
   他比我说得要好些。
   ```
