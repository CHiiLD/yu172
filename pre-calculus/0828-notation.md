# Notation
## 수체계
$\mathbb{N} =\{1, 2, 3 ...\}$ : 자연수
$\mathbb{Z} =\{... -2, -1, 0, 1, 2, 3 ...\}$ : 정수
$\mathbb{Q} =\{\frac{n}{m}| n, m\in\mathbb{Z} , m\ne0\}$ : 유리수
$\mathbb{Q}^c$ : 무리수
$\mathbb{R} = \mathbb{Q} \cup \mathbb{Q}^c$ : 실수
$\mathbb{C} = \{a+bi | a,b \gets  \mathbb{R}\}$
